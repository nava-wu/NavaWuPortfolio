# My Personal Portfolio

A portfolio to show my personal and professional projects, completed during my firmware development career and during my time at the University of Waterloo.

## Tech Stack

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with gradients, animations, and flexbox/grid layouts
- **Vanilla JavaScript**: Page navigation, scroll animations, and interactive elements
- **SVG**: Custom logo design

## Key Sections

- **Landing Page**: Hero section with tagline and key technologies
- **About Me**: Education, contact info, technical skills, and personal interests
- **Projects**: Showcase of major projects with detailed descriptions
- **Work Experience**: Professional timeline with roles and achievements
- **Project Detail Pages**: Deep dive into specific projects (example included)

## Responsive Breakpoints

- **Desktop**: 1200px+ (full layout)
- **Tablet**: 768px - 1199px (adjusted grid)
- **Mobile**: < 768px (single column, stacked navigation)

## Performance

- No external dependencies or frameworks
- Minimal JavaScript for fast load times
- Optimized CSS with modern layout techniques
- SVG logo for crisp display at any size
