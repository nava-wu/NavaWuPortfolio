App-Connected Smart Menstrual Cup

Sep 2024 - Mar 2025

Overview

This project aimed to create a smart menstrual cup that could communicate with a web application to provide real-time health insights. The device uses sensors to collect data and transmits it via Bluetooth to help users track their menstrual health more effectively.

Technical Implementation

The project went through multiple iterations, starting with an ESP32 Pico before transitioning to the STM32WB55 for better Bluetooth Low Energy support and lower power consumption. I wrote all firmware from scratch in C, utilizing STM32Cube tools for configuration and development.

Key Challenges

  • Hardware Transition: Moving from ESP32 to STM32 required rewriting the entire firmware stack and learning the STM32 ecosystem
  • Bluetooth Communication: Implementing reliable BLE communication with proper service and characteristic definitions
  • Custom PCB Integration: Ensuring firmware worked seamlessly with custom hardware design
  • Real-time Database: Integrating Firebase for cloud data storage and retrieval

Technology Stack

C STM32WB55 STM32Cube Tools Bluetooth Low Energy Firebase Web Application Custom PCB

Outcomes & Learning

This project taught me the importance of reaching out to experts and iterating on designs. I owned the entire firmware development process, web application software, and database integration, which gave me end-to-end product development experience. The final product is a polished, robust solution that demonstrates my ability to bring a complex embedded system from concept to reality.
